Self Evaluation

Skills I've Learned

Throughout my work in this program, I have learned lots about programming. I completely learned to program from no prior experience to being proficient in Python, C++, Javascript, and HTML. I also have moderate experience with SQL, NoSQL, and SQLite. I have also learned lots about secure coding, best practices, and coding concepts, like Object-Oriented Programming. My continued learning has contributed to reinforcement of previously learned skills as well as being able to learn more advanced concepts

In two courses, I learned about the agile approach and lots about the Scrum Framework. To reinforce the learning in this course, there were group assignments in which we adopted roles in the Scrum Framework, learning more about Scrum Master, Project Manager, Stakeholders, and Developers. This course also taught me how to effectively communicate with Stakeholders and to speak in non-technical terms so they can fully understand the work being done.

I learned various Data Structures and Algorithms including Binary Search Trees, Linked Lists and Vectors. Through this work I learned various algorithms like QuickSort, which helped me understand how the structures work and the best ways to apply each type of structure. I have learned about finding ways to reduce operation time using Big O notation and finding the most efficient solution.

I have been able to apply all that I have learned to engineer programs that incorporate security, databases, and algorithms. Properly weaving all of these concepts together has been something that has taken a while to fully understand and complete effectively. Though there are lots of different parts, being able to apply them all to make a working program has been one of the largest concepts I have learned.

By putting together this portfolio, I have been able to examine my previous work, and see aspects of my projects that excel, as well as fix areas that were lackluster. Overall, this portfolio has helped me realize the areas that I excel in, making myself more employable by emphasizing these skills.
