Course Sorting
Program

This is a C++ program to sort course objects, read from a CSV file, using a Binary Search Tree. The program allows the user to use their own files, loading and organizing their data. The program also will output the data in order, query the tree for a specific course, and deconstruct the tree should they wish to load a new file.
